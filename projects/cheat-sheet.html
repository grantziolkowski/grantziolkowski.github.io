CSS CheatSheet

CSS is the web language used for styling and formating of HTML. Normally you write CSS in an external stylesheet in its own file that you can link to from your HTML document. Below are some of the basic properties of CSS.

  • Selectors

    • CSS selectors allow you to select and manipulate HTML elements. CSS selectors are used to "find" (or select) HTML elements based on their id, class, type, attribute, and more.

    The Element Selector

    <p> This is a paragraph in your HTML </p>

    This is how your CSS file could look:

    p {
    color: red;
    }

    After formatting:

    This is a paragraph

    The id Selector

    The id selector uses the id attribute of an HTML element to select a specific element. An id should be unique within a page, so the id selector is used if you want to select a single, unique element. To select an element with a specific id, write a hash character, followed by the id of the element. The style rule below will be applied to the HTML element with id="para1":

    <p id="para1"> This is a paragraph in your HTML with the para1 id.</p>

    Example:

    #para1 {
    text-align: center;
    color: red;
    }

    After formatting:

    This is a paragraph in your HTML with the para1 id.

    The class Selector

    The class selector selects elements with a specific class attribute. To select elements with a specific class, write a period character, followed by the name of the class: In the example below, all HTML elements with class="center" will be center-aligned:

    <p id="center"> This is a paragraph in your HTML with the center class.</p>

    <p id="center"> This is another paragraph in your HTML with the center class.</p>

    Example

    .center {
    text-align: center;
    color: blue;
    }

    After formatting:

    This is a paragraph in your HTML with the center class.

    This is another paragraph in your HTML with the center class.

    Try it yourself You can also specify that only specific HTML elements should be affected by a class. In the example below, all <p> elements with class="center" will be center-aligned. Example:

    p.center {
    text-align: center;
    color: red;
    }

  • Class vs. IDs

    • Class are a grouping of many lines of code that you want to have the same formating.

    IDs are like classes but are unqiue to one object. You only want that part of the code to have that styling. They are like nicknames for that code.

  • Margin

    • Clear an area around an element. The margin does not have a background color. The top,right,bottom and left margin can be changed independently. And using Chrome dev tools makes them easlier to navigate. Here's an example using margin on an "indent" class we created. This will indent whatever you assign to that class 50 px from the left:


    .indent {
    margin-left: 50px;
    }

    Another example:


    .huge_indent {
    margin-left: 200px;
    }

  • Positioning

    • Positioning allows you to move sections/elements in your site around the wire frame of your site. There are four main types of positioning in CSS. Static, Fixed Positioning, Relative, and Absolute.

  • Hover

    • Hover lets you adjust the style of a link only for when your mouse is "hovering" over it. Make sure you're hovering over it. For example:

    www.google.com becomes: www.google.com
